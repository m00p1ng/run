# RUN

shortcut for execution file with extension

## Installation

put configuration in your `$HOME/.bashrc` or `$HOME/.zshrc`

```
source /path/to/your/run.sh
```

## Usage

```
$ run FILE_WHAT_YOU_WANT
```

### Support

- Python (\*.py)
- Golang (\*.go)
- C (\*.c)
- C++ (\*.cpp)
- Ruby (\*.rb)
- Shell (\*.sh)
- Haskell (\*.hs)
- Clojure (\*.clj)
- Elixir (\*.ex)
